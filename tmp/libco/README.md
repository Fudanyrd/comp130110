# Libco
A minimized coroutine library. 

Currently support:
<table>
  <tr>
    <td> proc_init </td>
    <td> create a runnable process </td>
  </tr>
  <tr>
    <td> proc_yield </td>
    <td> give up the cpu for another thread </td>
  </tr>
  <tr>
    <td> proc_exit </td>
    <td> thread itself stops execution </td>
  </tr>
</table>
